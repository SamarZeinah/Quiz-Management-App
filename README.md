# 🧠 Quiz App – Quiz Management System

A full-featured **Quiz Management System** built with **React (Vite + TypeScript)**, designed for both **instructors and students** to manage and participate in quizzes efficiently.

---

## 🚀 Overview

This application provides a complete platform where:

- 👨‍🏫 Instructors can manage quizzes, questions, groups, and students  
- 👩‍🎓 Students can join quizzes, attempt them, and view results  
- 📊 Real-time dashboards show performance and analytics  

---

## ✨ Features

### 🔐 Authentication System
- Login / Register (Student & Instructor)
- Forgot / Reset / Change Password
- JWT-based secure authentication
- Protected routes with role-based access

---

### 📊 Dashboard
- Top 5 upcoming quizzes
- Top students by average score
- Quick navigation to main modules

---

### 👨‍🎓 Student Management
- Add / Edit / Delete students  
- Assign students to groups  
- Track performance  

---

### 👥 Groups Management
- Create and manage groups  
- Assign or remove students  

---

### ❓ Questions Bank
- Add / Edit / Delete questions  
- Organize questions per quiz  

---

### 🧠 Quiz System
- Create and manage quizzes  
- Join & attempt quizzes  
- View completed quizzes & results  

---

### 📈 Results & Analytics
- Quiz performance tracking  
- Student score history  
- Visual insights (charts & summaries)  

---

## 🛠️ Tech Stack

- ⚛️ React 19 + Vite  
- 🟦 TypeScript  
- 🎨 Tailwind CSS + Shadcn UI  
- 🔄 React Router (HashRouter)  
- 🧾 Formik + Yup  
- 🌐 Axios (API calls)  
- 🔐 JWT Authentication  
- 🎯 Context API (State Management)  
- 🎛️ Radix UI + Lucide Icons
